Mobile safety

Mobile app permissions

Grant only required app permissions and remove stale access over time.

Core guidance

  • Allow camera, microphone, and location only when needed.
  • Audit background activity and data access for installed apps.
  • Revoke permissions for apps not used recently.

Implementation checklist

  1. Define data scope and purpose for this feature area.
  2. Apply least-privilege access and default-deny telemetry.
  3. Test export/delete behavior before release.
